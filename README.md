# leetcode
## [memo] how to test "write directly" in vscode-leetcode
- Enclose in ""
  - "1234"
  - ```
    1234
    ```
  - "[1,2,3]"
  - ```
    [1,2,3]
    ```
  - ""abc""
  - ```
    "abc"
    ```
- Use "\n" for line breaks
  - ""hello"\n"world""
  - ```
    "hello"
    "world"
    ```
  - "123\n456"
  - ```
    123
    456
    ```
